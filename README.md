This repository contains four data analytics tasks completed using Python as part of the InfynixTech Internship Program.

The projects demonstrate skills in:

- Data Visualization
- Sentiment Analysis

# Task 1: Web Scraping

## Objective
Extract data from a website and store it in a structured format.

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Files
- InfynixTech_WebScraping.py
- quotes_dataset.csv

## Features
- Extracts quotes and authors from a website
- Stores data in CSV format
- Creates a clean dataset for analysis

## Output
A dataset containing quotes and their respective authors.
