# InfynixTech Web Scraping Project

## Project Overview

This project demonstrates web scraping using Python. Data was extracted from a website using BeautifulSoup and Requests, then stored in a CSV dataset using Pandas.

## Technologies Used

* Python
* Requests
* BeautifulSoup
* Pandas

## Files

* InfynixTech_WebScraping.py : Python script for web scraping
* quotes_dataset.csv : Generated dataset containing quotes and authors

## Output

The project extracts quotes and their authors and saves them into a structured CSV file for further analysis.
